## Project Confluence Identity
```
CONFLUENCE_SPACE_KEY=                     # e.g. ORD — project space for pages
```

## Decision Records
```
SDR_CONFLUENCE_SPACE=                     # space key for SDRs (defaults to CONFLUENCE_SPACE_KEY)
SDR_PARENT_PAGE=Software Decision Records # parent page title under which SDRs are stored
ADR_PARENT_PAGE=Architecture Decisions    # parent page title under which ADRs are stored
```
